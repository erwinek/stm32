32L152CDISCOVERY and STM32L-Discovery FW Package update history

 

V1.0.3 / 30-April-2013

Main Changes

  • added project settings for 32L152C-DISCOVERY board
  • update for the latest version of Touch-Sensing Driver and Standard Peripheral Library

Contents

  • AN3413-STM32L1-Current_consumption_and_touch_sensing_demo V1.0.3 (release notes)
  • AN3964-STM32L1-Temperature_sensor_example V1.0.3 (release notes)
  • STM32L1xx Standard Peripherals Library Drivers (StdPeriph_Driver) V1.1.1 (release notes)
  • STM32 Touch-Sensing Driver V1.3.2 (release notes)

 
Development Toolchains

  • IAR EWARM version v6.50.3
  • Keil MDK-ARM  v4.60
  • Attolic TrueSTUDIO v4.0
  • TASKING VX-toolset for ARM v4.4

 

V1.0.2 / 14-September-2011

Main Changes

  • Renamed from "STM32L-DISCOVERY kit demonstration software source code" to "STM32L-Discovery Firmware Package"
  • Added firmware associated to AN3964
  • Update of STM32 Touch-Sensing Driver

 

Contents

  • AN3413-STM32L-Discovery_Current_consumption_and_touch_sensing_demo V1.0.2 (release notes)
  • AN3964-STM32L-Discovery_Temperature_sensor_example V1.0.0 (release notes)
  • STM32L1xx Standard Peripherals Library Drivers (StdPeriph_Driver) V1.0.0 (release notes)
  • STM32 Touch-Sensing Driver V0.1.3 (release notes)

 

Development Toolchains

  • IAR EWARM from version 6.21
  • Keil MDK-ARM from 4.21
  • Attolic TrueSTUDIO from V2.1.0

 

STLink Support

  • After installing EWARM 6.21, you should install the ST-LINK/V2 driver by running the ST-Link_V2_USB.exe from [IAR_INSTALL_DIRECTORY]\Embedded Workbench 6.0_2\arm\drivers\ST-Link\ST-Link_V2_USBdriver.exe
  • ST-LINK/V2 driver is installed automatically after installing Keil MDK-ARM 4.21.
  • ST-LINK/V2 driver is installed automatically after installing Attolic TrueSTUDIO V2.1.0
